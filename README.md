# HaloSync

**Beyond API — Travel Infrastructure as a Service**

HaloSync is a next-generation NDC SaaS platform that simplifies airline distribution and enables travel sellers to scale. We provide unified access to 12+ airline NDC APIs through a single integration point.

## What We Build

| Product | Description |
|---------|-------------|
| **PolarHub API** | Unified NDC API gateway — one integration, 12+ airlines |
| **Albus** | Platform control & admin — sales management, route control, promotions, settlement & reporting |
| **Luna** | B2B portal service — flight search, booking/ticketing, post-booking, full NDC content access |

## Open Source

These repositories demonstrate how OTAs can build flight booking products on top of the PolarHub NDC API:

| Repository | Description |
|------------|-------------|
| [whitelabel-client](https://github.com/HaloSync-Aggregator/whitelabel-client) | Multi-tenant flight booking SPA generator — Vite + React, AI-powered code generation |
| [whitelabel-middleware](https://github.com/HaloSync-Aggregator/whitelabel-middleware) | NestJS middleware gateway for PolarHub NDC API — auto-generated by Claude Code |
| [polarhub-mcp-client](https://github.com/HaloSync-Aggregator/polarhub-mcp-client) | AI flight booking chatbot via MCP — natural language to NDC API |
